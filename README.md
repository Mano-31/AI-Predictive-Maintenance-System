# 🏭 AI-Based Predictive Maintenance System

## 📌 Project Overview

This project uses Machine Learning to predict industrial machine failures before they happen.

The system analyzes machine sensor data and predicts whether the machine is working normally or has a failure risk.

---

## 🎯 Problem Statement

Unexpected machine failures cause production loss and high maintenance costs.

This project aims to build an AI-based predictive maintenance system that detects possible machine failures in advance.

---

## 🚀 Objectives

- Predict machine failure using sensor data
- Reduce unexpected downtime
- Improve maintenance planning
- Support smart manufacturing

---

## 📂 Dataset

Dataset:
AI4I 2020 Predictive Maintenance Dataset

Features:

- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear
- Machine type
- Failure types

---

## 🛠 Technologies Used

Programming:

- Python

Libraries:

- Pandas

- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost
- Streamlit
